# 输出格式参考

RAGFlow skill 响应的统一风格指南。
本参考适用于该 skill 的所有面向用户的输出。

## 格式选择矩阵

| 信息类型 | 格式 | 使用场景 |
|---------|------|---------|
| 多条目（3+）且含属性 | **表格** | 数据集列表、搜索结果 |
| 顺序步骤 | **有序列表** | 上传流程、操作步骤 |
| 功能/选项 | **无序列表** | 功能概览 |
| 结构化数据 | **JSON 代码块** | API 响应 |
| 文档内容 | **引用块** | 检索到的分块 |
| 单对象属性 | **定义列表** | 数据集详情 |
| 状态 | **emoji + 文字** | ✅ 完成、🟡 进行中、❌ 失败 |

## 常用格式

### 表格（3 条及以上）
```markdown
| 数据集 | 文档数 | 分块数 | 状态 |
|--------|--------|--------|------|
| delete | 4      | 53     | ✅   |
```
- 长 ID 缩写显示：`abc123...`
- 状态使用 emoji：✅ ❌ 🟡 ⚠️

### 无序列表
```markdown
- **上传文档**到数据集
- **启动解析**以生成分块
```
- 操作类条目以动词开头
- 最多 2 级缩进

### 有序列表
```markdown
1. 创建数据集
2. 上传文件
3. 启动解析
```
- 用于顺序操作步骤

### 状态图标
| 图标 | 含义 |
|------|------|
| ✅ | 成功 |
| ❌ | 失败 |
| 🟡 | 进行中 |
| ⚠️ | 警告 |
| ⬜ | 为空 |

## 响应模板

**列表操作：**
```markdown
📋 **数据集**（共 3 个）

| 名称 | ID | 状态 | 分块数 |
|------|----|------|--------|
| test | abc... | ✅ | 152 |
```

**搜索结果：**
```markdown
🔍 **结果**（找到 2 条）

| # | 来源 | 相似度 | 内容 |
|---|------|--------|------|
| 1 | doc.pdf | 85% | 摘要... |
```

**对象详情：**
```markdown
📊 **数据集详情**

**ID：** `1ce917df20e411f191a984ba59bc54d9`
**名称：** delete
**分块数：** 53
```
