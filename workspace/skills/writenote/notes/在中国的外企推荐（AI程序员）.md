# 在中国的外企推荐（AI程序员）

目标口径：在中国有公开可核验存在；对 AI/软件工程师相对友好；避开微软/谷歌等超大厂。

## 怎么用（最省事的投递法）
1. 先选 2–3 条主线：EDA/芯片软件、仿真/工业软件、企业软件/平台、安全、通信网络智能化。
2. 用同一组关键词混合中英检索 JD：Software Engineer / R&D / Platform / Backend / Python / C++ / Linux / Kubernetes；以及 Machine Learning / MLOps / Data Platform。
3. 优先找在华 R&D/工程团队岗位；对长期驻场、第三方合同主体、纯销售岗要谨慎。

## 个人行动清单（1 周内可完成）
1. 给每家公司建一个收藏夹：官网招聘页 + LinkedIn + 本地招聘平台同名检索。
2. 每家只投 1–2 个最匹配岗位，配套准备一份对应方向的项目/简历版本。
3. 如果目标偏平台/数据/ML Infra：优先补充一个可展示的部署或数据链路项目（K8s + 推理服务 或 ETL/特征管道）。
